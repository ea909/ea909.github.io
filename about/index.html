Software Development

I write software, both professionally and as a hobby. Most of my professional work has been in embedded systems, specifically small 8, 16, and 32 bit microcontrollers plus a bit of VHDL. In addition to embedded work, I've worked on a lot of desktop application software to interface with or operate custom hardware.

In my hobbies, my focus is on audio synthesis, game development, and some graphics programming.

Some stuff I have done:

  • C and C++
  • Microcontrollers, specifically Microchip's line.
  • Some VHDL (for Xilinx FPGAs).
  • Windows API wrangling, sockets, aynchronous IO, general systems programming.
  • OpenGL 3, custom GUI development, shaders, mostly 2D work.
  • Audio programming: mixing, sequencers, synthesizers, filters.
  • C#, GUI with Winforms and WPF.
  • Python, GUI with pyGTK, ML and text mining with pyTorch, numpy, scikit-learn
  • Lua scripting for various things.
  • HTML5 and CSS for a static site generator.

I am currently available for work (full time or contract); if interested, you can contact me at eric@badlydrawnsquirrel.net or eric@ericalzheimer.com.
