MStudio (working title)

MStudio is a chiptune-style music sequencer designed to be easy to use on touch devices. Most chiptune-esque sequencing software uses a tracker interface, which works well on a PC, but not on touchscreens. Touch devices are used without a dedicated keyboard, so a tracker interface becomes unergonomic.

Also, most music software on iOS/Android have UIs inspired by previous PC piano roll interfaces. These piano rolls work well on PC, where you can use left and right clicking plus modifier keys to express many actions. On mobile, translating the same set of actions often results in some painful, RSI inducing arrangement of mode switching buttons, tapping, long pressing, etc.

With MStudio, I will be making an interface that can be used efficiently using only single taps/clicks, with no mode switching. Target platforms are Linux/Windows/iOS/Android.

An OpenGL style C interface and UI toolkit for the FTDI FT800 graphics chip. By using an immediate mode GUI (IMGUI), it allows 8 bit microcontrollers to drive a graphical touch screen UI with minimal RAM usage. (GPL3 License)
